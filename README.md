# Analytic database backup

A brief description of the project.

## Installation

> install google-cloud-sdk

## Run
> bash start.sh


### Execution method
- Create .env file with environment variable
- Login GSC serivce with authorization
- Run backup
- Backup file saved in **backup/log.txt**
